# ICMP DoS Incident Response Report

This project documents the analysis and response to a simulated Denial of Service (DoS) attack
completed as part of the **Google Cybersecurity Analyst Certificate** program.

The scenario involved an ICMP packet flooding attack that disrupted internal network services
due to a misconfigured firewall. The project focuses on incident analysis, response actions,

- Network monitoring and detection
